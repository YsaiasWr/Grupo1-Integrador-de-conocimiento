# 🧑‍💼 Sistema de Gestion de Recursos Humanos

Proyecto desarrollado como parte del **Integrador de Conocimiento**.  
Este sistema permite administrar la informacion de empleados, departamentos y cargos dentro de una empresa.

---

## 📌 Objetivo del proyecto

Desarrollar una aplicacion que facilite el manejo de los datos del personal, permitiendo:

- Registrar empleados  
- Consultar informacion  
- Modificar registros  
- Organizar la estructura de la empresa  

---

## ⚙️ Funcionalidades

✔️ Registro de empleados  
✔️ Edicion y eliminacion de empleados  
✔️ Gestion de departamentos  
✔️ Gestion de cargos  
✔️ Asignacion de empleados a departamentos y cargos  

---
